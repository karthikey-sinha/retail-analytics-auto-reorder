
# Project Title

**Intelligent Retail Sales & Inventory Analytics System**


#  Short Description / Purpose

An end-to-end retail analytics project that simulates a real-world Superstore Management System using synthetic data generation and advanced Exploratory Data Analysis (EDA).

The project integrates automated inventory reordering logic with sales and profitability analysis to replicate how modern retail businesses monitor performance and optimize operations.

#  Data Source

The dataset is **custom-generated using Python (Faker library)** instead of using publicly available datasets.

* 1000 transaction records
* 25 structured business columns
* Includes sales, profit, customer segmentation, geography, supplier data, and inventory metrics
* Covers a 2-year simulated transaction timeline

This approach demonstrates data modeling capability rather than only analysis skills.

# Business Problem

Retail businesses often struggle with:

* Overstocking or stockouts
* Poor visibility into regional performance
* Discount strategies reducing profitability
* Lack of integration between sales data and inventory control

Without structured analysis, decision-making becomes reactive instead of data-driven.



#  Goal of the Dashboard

The dashboard aims to provide:

* Region-wise sales & profit visibility
* Category performance comparison
* Customer segment contribution analysis
* Discount vs Profit impact tracking
* Real-time identification of low-stock items
* Payment behavior distribution

It enables management to move from operational reporting to strategic optimization.
